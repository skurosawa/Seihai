# Seihai

Seihai は、思考を整えるためのミニマルな iOS アプリです。

入力 → 整理 → 行動

機能を増やすためのアプリではありません。  
体験の完成度を高めるためのプロジェクトです。

---

## コンセプト

Seihai は、3つのステップで思考を整えます。

1. 入力する
2. 並べ替える
3. 次の一手を決める

余計な装飾はありません。  
静かで、統一感のある体験を目指しています。

---

## 技術構成

- SwiftUI
- MVVM
- iOS 17+
- ローカル保存（UserDefaults）
- ネイティブハプティクス

---

## 設計思想
