# Seihai

Seihai は、思考を整えるためのミニマルな iOS アプリです。

**入力 → 整理 → 行動**

機能を増やすためのアプリではありません。  
体験の完成度を高めるためのプロジェクトです。

---

## コンセプト

Seihai は、3つのステップで思考を整えます。

1. 入力する  
2. 並べ替える  
3. 次の一手を決める  

余計な装飾はありません。  
静かで、統一感のある体験を目指しています。

---

## 行動生成ロジック

行動は常に1つだけ提示されます。

- 思考の中から最も「行動らしい」項目をスコア選択
- 不要な要約や自動改変は行わない
- 思考が変わると行動は無効化される
- 行動画面表示時に必要な場合のみ生成
- 閾値未満の場合は固定テンプレートを表示

Seihai は、提案しすぎません。

---

## 共有仕様

共有は Markdown 固定形式です。

```markdown
## 行動

{action}

## 整理

- {item1}
- {item2}
```

- 空セクションは出力しない
- 自動改変しない
- 絵文字を付与しない
- 共有成功時のみトースト表示
- キャンセル時は何も表示しない

## 技術構成

- SwiftUI
- MVVM
- Observation (@Observable)
- iOS 17+
- ローカル保存（UserDefaults Snapshot + Debounce）
- ネイティブハプティクス
- ultraThinMaterialベースUI
- フラットデザイン App Icon

---

## UX思想

Seihai の判断基準はひとつです。

> これはネイティブとして気持ちいいか？

- 機能を増やさない
- 迷わせない
- 行動は1つだけ
- 強調は 余白 → 階層 → 微細ボーダー → 最小影 → 色（最後）
- 過度なアニメーションをしない
- グラデーションを使わない

静かな完成度を目指しています。
